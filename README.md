AI DDR Report Generator

Overview

AI-powered DDR (Detailed Diagnostic Report) Generator that processes Inspection Reports and Thermal Reports and generates structured client-ready reports.

Features

- PDF Upload
- Text Extraction from Reports
- AI-powered DDR Generation using Gemini
- Missing Information Detection
- Structured Report Generation
- Download Generated Report

Tech Stack

- Python
- Streamlit
- Google Gemini API
- PyMuPDF
- Python Dotenv

Workflow

1. Upload Inspection Report PDF
2. Upload Thermal Report PDF
3. Extract text from both documents
4. Generate DDR using Gemini
5. Download generated report

Future Improvements

- Image Extraction from PDFs
- DDR Export to DOCX/PDF
- Severity Scoring Dashboard
- Multi-report Processing
